CSS

Вы, наверняка, уже заметили, что сайт преобразился: Появилось выравнивание, изменился шрифт и т.д.. Причиной этому является один из модулей HTML под названием CSS.

CSS — формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
Преимущественно используется как средство описания, оформления внешнего вида веб-страниц, написанных с помощью языков разметки HTML и XHTML, но может также применяться к любым XML-документам, например, к SVG или XUL.

Назначение CSS – отделять то, что задает внешний вид страницы, от ее содержания. Если документ создан только с использованием HTML, то в нем определяется не только каждый элемент, но и способ его отображения (цвет, шрифт, положение блока и т. д.). Если же подключены каскадные таблицы стилей, то HTML описывает только очередность объектов. А за все их свойства отвечает CSS. В HTML достаточно прописывать класс, не перечисляя все стили каждый раз.

Подключение CSS

Самый простой и удобный способ использования CSS - подключение отдельным файлом. Тоесть весь дизайн стрницы будет храниться не в ней самой, забивая место и усложняя код, а отдельно.

Для этого необходимо добавить данную строчку мужду тегами <head>:

<link rel="stylesheet" href="css/style.css>

После этого файл style.css, какторый должен находиться в одной папке с файлом HTML, станет отвечать за стиль сайта.

Кстати, для созданя элементов, подобных примеру кода для привязки, можно использовать классы.

Классы CSS

Классы в CSS - инструмент, который разработчиками используется очень часто. Классы позволяют задавать одному тегу различные стилевые правила. Давайте сразу рассмотрим на примере как их использовать. Но сначала проясним синтаксис селектора класса:

.Имя класса { свойство1: значение; свойство2: значение; ... }
В селекторе после имени тега через точку пишется имя класса. Имя класса разработчик придумывает сам. Имя класса должно начинаться с латинской буквы, может содержать дефис (-) и знак подчёркивания (_).
В коде HTML для указания связи тега с классом CSS в теге прописывается атрибут class="Имя класса".

Пример кода:

.main {
background-color: white;
}

Здесь можно увидеть, что некий объект класса main приобретает белый цвет фона.

Особенность классов заключается в том, что их можно присвоить множеству элементов

Сам же класс задаёт командой class. Она может быть применена к блочным элементам (div).

Пример синтаксиса:

<div class='main'></div>

ID CSS

В HTML есть такой атрибут id="". Он предназначен для присвоения тегам уникальных имён, которые используются для доступа к тегам при написании сценариев на JavaScript.
Но также они могут быть использованы и в CSS. С их помощью можно присваивать стили тегам по их уникальному id.

Значение id должно быть уникальным, на одной странице может встречаться только один раз и должно содержать хотя бы один символ. Значение не должно содержать пробелов.

Синтаксис применения селектора идентификатора следующий:

#Имя ID { свойство1: значение; свойство2: значение; ... }

Обращение к тегам

Также, помимо обращения к классу объекта или его ID, можно обратиться к типу объекта. Например объект типа body. Это тег. При обращении к ниму через css изменения коснутся срузу всей страницы. Можно обратиться к объектоам типа div. Тогда стиль всех объектов типа div будет изменён.

Пример синтаксиса:

Тег { свойство1: значение; свойство2: значение; ... }

Свойства CSS

Подведём итоги

CSS — один из важнейших модулей HTML, который позволяет изменять и корректировать внешний вид сайта. С его помощью можно изменять шрифт сайта, менять расположение элементов, добавлять элементы оформлния и т.п.. Без CSS невозможно представить современный интернет.
